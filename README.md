## Installation
1. install node (recommended node version 16.0.0)
2. Download or clone repository
3. `npm install --force` to install the required npm packages to run
4. `npm start`

## Available Scripts
In the project directory, you can run:

### `npm start`
### `npm test`
### `npm run build`
